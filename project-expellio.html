Expellio (Gestión de Finanzas Inteligente)

Expellio es una aplicación web y móvil para la gestión financiera personal y de parejas. Permite a los usuarios registrar y categorizar ingresos y gastos, configurar presupuestos mensuales con alertas de consumo, establecer metas de ahorro conjuntas (como fondos de emergencia) y dividir deudas equitativamente. Además, integra procesamiento por IA que automatiza la carga de transacciones mediante la transcripción de notas de voz y el escaneo fotográfico de recibos físicos.

Contribuciones técnicas clave

Aislamiento de Multi-Inquilino (Multi-Tenancy): Implementé un diseño estricto de base de datos donde toda la información financiera está aislada de forma lógica a nivel de consulta mediante un identificador de hogar (`householdId`), asegurando privacidad y seguridad de los datos.

Seguridad contra Ataques de Temporización (Timing Attacks): Previne la enumeración de usuarios implementando una simulación de retardo computacional con `bcrypt.compare` contra hashes ficticios cuando un correo electrónico consultado no existe en el sistema.

Rotación de Token de Refresco (RTR) con Estado: Programé un flujo de autenticación con doble JWT que audita los tokens en una base de datos PostgreSQL, previniendo ataques de replay mediante la invalidación automática de reintentos con identificadores de token (`jti`) usados.

Registro de Auditoría Append-Only: Diseñé un motor de log de auditoría inmutable a nivel de base de datos a través de PostgreSQL, garantizando trazabilidad total de operaciones críticas y bloqueando actualizaciones o eliminaciones de auditoría.

Procesamiento en Segundo Plano de IA: Modelé registros de auditoría y colas de tareas asíncronas para el procesamiento de IA (`AiJob`) de notas de voz, imágenes de tickets e informes mensuales dinámicos.
