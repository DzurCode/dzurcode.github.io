## 🛠 Technologies & Tools

The portfolio itself is built with performance, premium aesthetics, and maintainability in mind, without relying on heavy frontend frameworks:

- **HTML5 & Semantic Structure**
- **Vanilla JavaScript**: For lightweight interactions (scroll reveals, modal image galleries, smooth navigation).
- **Tailwind CSS (v3)**: Used as the core utility-first styling engine, configured with custom design tokens for typography, glowing gradients, and glassmorphism effects.
- **Devicon**: For injecting high-quality SVG technology icons dynamically.

## 💻 Running Locally

To run this project on your local machine and compile the Tailwind styles, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/DzurCode/portfolio.git
   cd portfolio
   ```

2. Install the necessary development dependencies (Tailwind CSS):
   ```bash
   npm install
   ```

### Development

To compile the Tailwind CSS file, run the following build script:

```bash
npm run build:css
```

You can then open the `index.html` file directly in your web browser, or use a local development server like VS Code's **Live Server** extension for a better development experience with hot reloading.

## 🎨 Design & Architecture Highlights

- **Premium Aesthetics**: Features a dark mode interface with vibrant glowing orb backgrounds, blurred glassmorphism cards, and subtle CSS micro-animations.
- **Responsive Layout**: Uses CSS Grid and Flexbox to seamlessly adapt from large desktop monitors down to mobile screens.
- **Custom CSS Engine**: The `/assets/css/input.css` file contains structural styling, animation keyframes, and custom Tailwind `@layer` directives.
- **Agent-Driven Development (ADD)**: Built and maintained leveraging AI tools like Antigravity, GitHub Copilot, and Claude to ensure clean code and rapid iteration.
