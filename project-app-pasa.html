Logros

Contribuciones técnicas clave

Desarrollé una billetera digital multiplataforma (iOS + Android) para depósitos, retiros, envíos y conversión de criptomonedas en el ecosistema Stellar Blockchain, aplicando Clean Architecture + BLoC para máxima modularidad y testabilidad.

Implementé el flujo completo de cuentas Stellar: creación, consulta de balances, firma de transacciones con Horizon y estándares SEP-10 (auth con anchors) y SEP-24 (depósitos/retiros).

Implementé soporte offline-first con caché local (sqflite/Hive) para consulta de balances e historial sin conexión.

Construí un sistema de multisig para seguridad avanzada y recuperación de cuentas.

Configuré el pipeline CI/CD completo con Codemagic para builds automáticos, firma de certificados y publicación en Google Play Store y Apple App Store.

Garanticé calidad con pruebas unitarias (flutter_test, bloc_test), de widgets y de integración (integration_test + mockito), alcanzando cobertura en todos los BLoCs y repositorios críticos.

Refactoricé la API backend en Node.js + TypeScript y desarrollé un microservicio para envío de OTP por email y SMS.
